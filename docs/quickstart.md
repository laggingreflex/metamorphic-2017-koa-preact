# Quickstart

## Requirements

* Node

* MongoDB

* Redis

## Quickstart

* Clone this repo.

* Install dependencies.

  ```
  npm install
  ```

* Build

  ```
  npm start build
  ```

* Run

  ```
  npm start run
  ```

  Run in development mode (watch/hot-reload):

  ```
  npm start runDev
  ```


* Configure settings, like port/API keys/etc.:

  ```
  npm start config
  ```
